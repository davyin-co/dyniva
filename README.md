# 介绍
dyniva是基于acquia公司的lightning发行版开发而来的Drupal发行版。
# 功能特色
1. 精选的drupal模块
1. 组件化建站
1. 适合国情的后台管理界面
1. 集成常见的应用
1. 内容编辑流程
1. 搜索增强
1. 媒体增强
1. SEO
1. ... ...
